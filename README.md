📌 Sobre o projeto

O BBQ BRO é um cardápio digital desenvolvido para apresentar produtos de forma moderna e atrativa, focado principalmente em hamburguerias e restaurantes.

Este projeto é apenas front-end, ideal para estudos, portfólio ou base para integração futura com back-end.

🚀 Funcionalidades
📱 Layout responsivo
🍔 Exibição de produtos (hambúrgueres, acompanhamentos e bebidas)
📝 Descrição detalhada dos itens
💰 Exibição de preços
🎨 Design moderno com tema dark
🛒 Botão de pedido (simulação)
🛠️ Tecnologias utilizadas
HTML5
CSS3
Bootstrap 5
